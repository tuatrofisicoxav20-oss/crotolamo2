# Crotolamo Runtime v12 Task Planner

Agrega planificador técnico.

## Comandos

```text
plan mejorar extractor de huevonitis
plan revisar precisión de tletl
plan limpiar estructura de crotolamo
planes
planes recientes
```

## Qué hace

- Detecta proyecto por objetivo
- Usa inspector/indexador
- Genera pasos seguros
- Clasifica riesgo de comandos
- Guarda planes JSON en `data/task_plans/`
- No ejecuta cambios
